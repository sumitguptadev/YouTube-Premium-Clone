# 🎬 YouTube Premium Clone

A modern and fully responsive **YouTube Premium Landing Page Clone** built using **HTML & CSS**.  
This project recreates the premium look and feel of YouTube Premium with clean layouts, responsive sections, and smooth frontend structure.

---

## 🚀 Features

- 🎨 Modern YouTube Premium inspired UI
- 📱 Fully Responsive Design
- ⚡ Clean and smooth layout structure
- 🖼️ Custom assets & images used
- 💻 Built using only HTML & CSS
- 🧩 Beginner-friendly project structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Assets (Images/Icons)
